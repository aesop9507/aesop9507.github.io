# 랜섬웨어의 퇴조, 디지털 기생충의 부상 - 현대 사이버 공격의 은밀한 전략 변화

> **요약**: 지난 10년간 랜섬웨어가 사이버 위협의 대명사였지만, 2025년의 공격 데이터는 명백한 변화를 보여줍니다. 공격자는 더 이상 파괴적인 암호화를 최우선으로 하지 않습니다. 대신 장기적, 은밀한 접근을 목표로 하는 "디지털 기생충(Digital Parasite)" 모델이 떠오르고 있습니다. Picus Labs의 Red Report 2026이 밝혀낸 새로운 위협 환경을 분석합니다.

---

## 1. 랜섬웨어 시그널의 약화

### 38% 감소한 데이터 암호화 공격

지난 10년 동안 랜섬웨어 암호화는 사이버 리스크의 가장 명확한 신호였습니다. 시스템이 잠기고 운영이 중단되면, 침해는 부정할 수 없는 사실이 되었죠.

하지만 2025년의 데이터는 다른 이야기를 들려줍니다:

- **Data Encrypted for Impact (T1486)**: 2024년 21.00%에서 2025년 12.94%로 **38% 감소**
- 이 감소는 공격자 능력의 저하가 아닌 **전략적 전환의 결과**

### 왜 암호화를 포기했나?

공격자는 시스템을 잠가 돈을 강요하는 대신, 다음과 같은 방식으로 시스템을 **계속 가동**하면서:

- 민감 데이터를 조용히 탈취
- 자격증명과 토큰을 수집
- 장기간 환경에 잠복
- 나중에 방해가 아닌 협박을 통해 압력을 가함

이 변화의 시사점은 명확합니다: **영향력은 더 이상 잠긴 시스템으로 정의되지 않습니다. 공격자가 얼마나 오랫동안 호스트 시스템 내에서 탐지되지 않고 접근을 유지할 수 있느냐가 핵심입니다.**

> "적의 비즈니스 모델은 즉각적인 방해에서 장기 생존 접근으로 이동했습니다." — Picus Red Report 2026

## 2. 자격증명 도굴이 새로운 제어 플레인이 되다

### 4건 중 1건의 공격이 자격증명 탈취

공격자가 장기적, 은밀한 지속성으로 전환함에 따라, **아이덴티티**가 가장 신뢰할 수 있는 제어 경로가 되었습니다.

Red Report 2026에 따르면:

- **Credentials from Password Stores (T1555)**: 지난 1년간 관찰된 모든 공격의 **23.49%**에 등장
- 4건 중 1건은 브라우저, 키체인, 비밀번호 관리자에서 저장된 자격증명을 직접 추출

### 소음 없는 진입

과거의 노이즈한 자격증명 덤핑이나 복잡한 익스플로잇 체인에 의존하는 대신, 공격자는:

1. 저장된 자격증명을 직접 추출
2. 유효한 자격증명 확보 후 권한 상승과 횡적 이동(Lateral Movement)은 네이티브 관리 도구만으로 충분

현대의 멀웨어 캠페인은 디지털 기생충처럼 행동합니다:

- 경고 없음
- 충돌 없음
- 명백한 지표 없음
- 그저 **기이한 침묵**만이 존재

## 3. 최상위 ATT&CK 기술의 80%가 은폐에 집중

### 전례 없는 은폐 기술 집중도

MITRE ATT&CK® 프레임워크의 폭넓은 기술에도 불구하고, 실제 멀웨어 활동은 은폐, 지속성, 은밀한 명령 제어를 우선하는 소수의 기술 집합에 계속 집중됩니다.

Red Report 2026은 **Picus Labs가 기록한 역대 가장 높은 은폐 중심 전술 집중도**를 보여줍니다:

**최상위 10개 MITRE ATT&CK 기술 중 8개가 은폐, 지속성, 정찰에 집중**

| 기술 | MITRE ID | 목적 |
|------|----------|------|
| Process Injection | T1055 | 신뢰할 수 있는 시스템 프로세스 내에서 실행 |
| Boot or Logon Autostart Execution | T1547 | 재부팅과 로그인에서 생존 보장 |
| Application Layer Protocols | T1071 | 정상 웹/클라우드 통신에 섞여 "속삭이는 채널" 제공 |
| Virtualization and Sandbox Evasion | T1497 | 분석 환경 탐지 및 실행 거부 |

### 결과: 합법처럼 보이는 프로세스가 합법적인 도구를 사용하여 신뢰할 수 있는 채널에서 조용히 운영됩니다.

- **서명 기반 탐지**: 이 환경에서 어려움 겪음
- **행동 분석**: 정상적으로 보이도록 고안된 불법 활동 식별에 점점 더 중요

**암호화가 공격을 정의했던 시대는 갔습니다. 이제 은폐가 성공을 정의합니다.**

## 4. 자기 인식 멀웨어: 분석을 거부하다

### 비활성조차 회피 기술

은폐가 성공의 주요 척도가 되면, 탐지 회피만으로는 충분하지 않습니다. 공격자는 수비자가 악의적인 행동을 관찰하는 데 의존하는 도구를 피해야 합니다.

Red Report 2026은 이를 **Virtualization and Sandbox Evasion (T1497)**의 상승에서 명확히 보여줍니다. 이 기술은 2025년에 공격자 전술의 최상위 티어에 진입했습니다.

### LummaC2: 마우스 움직임 분석

현대 멀웨어는 행동하기 전에 **어디에 있는지 평가**합니다:

- 단순한 아티팩트 체크에 의존하지 않음
- 실행 컨텍스트와 사용자 상호작용을 평가하여 실제 환경인지 판단

**LummaC2 사례**:
- 기하학을 사용해 마우스 움직임 패턴 분석
- 유클리드 거리와 커서 각도를 계산하여 **인간 상호작용과 샌드박스 환경의 선형 움직임 구분**
- 조건이 인위적일 경우 의도적으로 실행 억제하고 조용히 대기

이 동작은 공격자 로직의 더 깊은 변화를 반영합니다:

1. 샌드박스 환경에서 더 이상 자신을 드러내지 않음
2. 설계에 의해 활동을 보류
3. 실제 프로덕션 시스템에 도달할 때까지 휴면 상태 유지

### 무위(無爲)가 핵심 회피 기술

은폐와 지속성이 지배하는 생태계에서 **비활성 그 자체가 핵심 회피 기술이 되었습니다.**

## 5. AI의 현실: 혁명이 아닌 진화

### AI 기대 vs 현실

공격자가 점점 더 적응적 행동을 보이면서, 인공지능이 이 그림에서 어떤 역할을 하는지 자연스럽게 궁금해집니다.

Red Report 2026 데이터는 **신중한 답변**을 제안합니다:

- 널리 퍼진 AI가 멀웨어 환경을 재편할 것이라는 광범위한 추측에도 불구하고
- 2025년 데이터셋에서 **AI 기반 멀웨어 기술의 의미 있는 증가는 관찰되지 않음**

### 여전히 지배적인 전통 기술

대신 가장 널리 퍼진 동작은 여전히 익숙한 것들입니다:

- Process Injection
- Command and Scripting Interpreter

이 기술들이 실제 침해를 계속 지배한다는 점은 **공격자가 현대 방어를 우회하는 데 고급 AI를 필요로 하지 않음**을 강화합니다.

### LLM API의 제한적 사용

일부 멀웨어 패밀리는 대규모 언어 모델(LLM) API 실험을 시작했지만, 사용 범위는 여전히 제한적입니다:

- 관찰된 경우에서 LLM 서비스는 주로 사전 정의된 명령을 검색하거나 편리한 통신 계층 역할
- 구현은 효율성을 개선하지만
- **공격자 의사결정이나 실행 논리를 근본적으로 변경하지 않음**

### 결론

현재까지 데이터는 AI가 기존 전술에 **흡수되는 것이지 재정의하는 것이 아님**을 보여줍니다:

- 디지털 기생충의 메커니즘은 변하지 않음
- 자격증명 도굴, 은밀한 지속성, 신뢰할 수 있는 프로세스 악용, 더 길어지는 체류 시간

공격자는 급진적 새 기술을 발명해서 이기는 것이 아닙니다. **그들은 더 조용해지고, 더 인내심이 강해지며, 합법적 활동과 구별하기 점점 더 어려워짐으로써 이기고 있습니다.**

## 6. 기본으로 돌아가기: 다른 위협 모델을 위해

### 변화하는 목표, 변하지 않는 전술

이 보고서를 매년 작성해 왔기에, 많은 같은 전술이 해마다 나타나는 지속적인 추세를 봅니다. **근본적으로 변화한 것은 목적(objective)입니다.**

현대 공격은 우선순위를 둡니다:

1. **보이지 않게 유지**
2. **신뢰할 수 있는 아이덴티티와 도구 악용**
3. **방어 조용히 비활성화**
4. **시간이 지남에 따라 접근 유지**

### 수비자의 새로운 초점

현대 보안 기본으로 **강화**하고:

1. **행동 기반 탐지** (Behavior-based detection)
2. **자격증명 위생** (Credential hygiene)
3. **지속적인 적대적 노출 검증** (Continuous Adversarial Exposure Validation)

조직은 다음에 덜 초점을 맞추고:

- 드라마틱한 공격 시나리오

오늘 실제로 성공하는 위협에 더 초점을 맞춰야 합니다.

## 7. 결론: 디지털 기생충에 대비하세요

### 침묵하는 위협이 더 위험하다

랜섬웨어 헤드라인이 여전히 뉴스 사이클을 지배하지만, Red Report 2026은 **실제 리스크가 침묵하고 지속적인 타협**에 더 많이 있음을 보여줍니다.

Picus Security는 가장 시끄러운 것뿐만 아니라 **공격자가 지금 실제로 사용하는 특정 기술**에 대한 방어를 검증하는 데 집중합니다.

### 핵심 인사이트

1. **암호화 위협 감소**: Data Encrypted for Impact가 38% 감소했지만, 이는 공격이 줄어든 것이 아니라 전략이 바뀐 것
2. **자격증명 도굴 부상**: 모든 공격의 23.49%가 저장된 자격증명 탈취에 집중
3. **은폐 최우선**: 최상위 ATT&CK 기술의 80%가 은폐, 지속성, 정찰에 집중
4. **자기 인식 멀웨어**: 샌드박스를 감지하고 실행을 거부하는 고도화된 회피 기술
5. **AI는 도구일 뿐**: 혁명이 아닌 진화. 기존 전술이 여전히 지배
